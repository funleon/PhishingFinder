# 釣魚郵件來找碴 (Phishing Finder) 

**Design by 資訊處資通安全管理部 石瑞慶 with AI  |  Last Updated: 2026/02/05**

這是一個旨在提升資安意識的互動式網頁遊戲。玩家需要在一封模擬的釣魚郵件中，找出 5 個可疑的特徵（如寄件者網域錯誤、製造緊急感、可疑連結等）。遊戲結束後，玩家可輸入姓名，系統將生成專屬的「資安意識結業證書」供預覽與下載。

## 📌 程式目的
- **教育訓練**：透過模擬真實情境，幫助員工或一般大眾識別社交工程攻擊。
- **互動學習**：相較於傳統宣導簡報，透過「找碴」遊戲增加參與感。
- **成果驗收**：提供個人化的結業證書下載，作為完成訓練的證明。

## 🚀 功能特色
- **隨機題庫**：每次遊戲隨機載入不同的釣魚郵件情境。
- **精準偵測**：點擊郵件不同區域（標題、內文、簽名檔）會觸發對應的提示。
- **即時回饋**：找到疑點時給予正向回饋並說明原因。
- **證書系統**：
  - **即時預覽**：輸入姓名後可立即預覽證書樣貌。
  - **跨平台支援**：完美支援 PC、平板 (iPad) 與手機瀏覽器。
  - **iOS 相容優化**：針對 iOS/Safari 系統特別優化，解決圖片儲存受限問題（支援長按「加入照片」）。
  - **自動檔名**：電腦版下載時會自動以「資安證書_姓名.png」命名。
- **RWD 設計**：響應式排版，適應各種螢幕尺寸。

## 🛠️ 部署方式
本專案為 **純前端 (Pure Front-end)** 架構，無需後端伺服器 (Database/Node.js/Python) 即可運作。

### 方法 1: 直接開啟 (最簡單)
直接雙擊點開 `index.html` 檔案即可在瀏覽器中執行。

### 方法 2: 本地伺服器 (推薦)
使用 VS Code 的 Live Server 插件或是 Python 快速啟動：
```bash
# Python 3
python -m http.server 8000
```
然後打開瀏覽器存取 `http://localhost:8000`。

### 方法 3: 網頁伺服器部署
將整個資料夾上傳至任何靜態網頁空間 (如 GitHub Pages, Netlify, Apache/Nginx 的 `www` 目錄) 即可。

## 🎮 使用方式
1. **開始遊戲**：點擊首頁的「開始挑戰」按鈕。
2. **尋找疑點**：仔細閱讀畫面右側的電子郵件，用滑鼠點擊你覺得可疑的地方 (共 5 處)。
   - 例如：寄件者信箱拼錯、威脅性的語氣、不明連結等。
   - 點擊正確會得分並顯示說明；點錯或重複點擊會有提示。
3. **查看結果**：找完所有疑點後，會顯示最終得分。
4. **領取證書**：
   - 輸入您的 **姓名**。
   - 點擊 **「預覽證書」** 按鈕。
   - **電腦版**：點擊預覽視窗右上角的下載圖示 (⬇️) 即可儲存圖片。
   - **手機/平板 (iOS/Android)**：直接 **長按** 預覽的證書圖片，選擇「加入照片」或是「下載影像」即可儲存。

## 📝 開發注意事項與已知限制
- **iOS 圖片儲存**：為了符合 iOS Safari 的嚴格安全性限制，在 iPad/iPhone 上產生證書時，會自動移除背景浮水印，以確保圖片能被系統正確識別並儲存至相簿。
- **題庫修改**：若要新增題目，請編輯 `js/data.js`。
- **Logo 更換**：目前證書 Logo 採 Base64 編碼嵌入於 CSS/HTML 中，若需更換可直接修改 `img` 標籤的 `src` 或相關 CSS。
- **字型**：證書採用「微軟正黑體」或系統預設無襯線字體。
