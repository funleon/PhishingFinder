# 釣魚郵件來找碴 (Phishing Finder)

這是一個純前端的互動式網頁遊戲，旨在透過模擬真實的釣魚郵件場景，提升使用者對資訊安全的警覺性與辨識能力。

## 🎯 專案特色

*   **隨機出題**：系統內建多種郵件場景，每次遊戲隨機抽選，保持新鮮感。
*   **熱點偵測**：使用者需找出郵件中的可疑之處（如偽造的寄件者、釣魚連結等），點擊正確即可得分。
*   **即時回饋**：點擊後提供即時的教育資訊，解釋為何該處可疑。
*   **結算與證書**：遊戲結束後會顯示得分，並可輸入姓名生成精美的「資安意識結業證書」圖片下載保存。
*   **純前端架構**：無需後端伺服器，部署簡單，支援離線執行。

## 🚀 部署與執行方式

由於本專案採用純 HTML/CSS/JS 開發，無須安裝 Node.js 或其他 runtime 環境。

### 方法一：直接開啟

1.  下載或 Clone 本專案到您的電腦。
2.  進入專案資料夾。
3.  雙擊 `index.html` 檔案，使用您喜歡的瀏覽器（建議使用 Chrome, Edge, Firefox, Safari 等現代瀏覽器）開啟即可開始遊戲。

### 方法二：使用 Local Server (建議)

為了獲得最佳體驗（避免部分瀏覽器對 `file://` 協議的安全性限制），建議透過簡易的 Local Web Server 執行：

*   **VS Code 使用者**：安裝 "Live Server" 套件，對著 `index.html` 右鍵點擊 "Open with Live Server"。
*   **Python 使用者**：在專案根目錄執行 `python -m http.server`，然後瀏覽器開啟 `http://localhost:8000`。

## 🎮 操作說明

1.  **開始遊戲**：開啟網頁後，點擊歡迎畫面上的「開始挑戰」按鈕。
2.  **尋找疑點**：觀察畫面中的模擬郵件，共有 **5 個** 可疑之處。
3.  **點擊偵測**：當您發現可疑點（例如怪異的 Email 地址、拼錯的網域、可疑的按鈕），請直接點擊該區域。
    *   ✅ **正確**：顯示提示視窗，解釋安全觀念，並獲得分數。
    *   ❌ **錯誤**：無反應或顯示錯誤提示（視設定而定）。
4.  **完成挑戰**：找完所有疑點或遊戲結束後，查看最終得分。
5.  **下載證書**：輸入您的姓名，點擊「下載證書」按鈕，即可將成績證明存為圖片。

## ⚙️ 自定義題庫

若需修改或新增郵件場景，請編輯 `js/data.js` 檔案。該檔案包含題庫的 JSON 資料結構，您可以依照現有格式新增新的郵件內容與熱點座標。
